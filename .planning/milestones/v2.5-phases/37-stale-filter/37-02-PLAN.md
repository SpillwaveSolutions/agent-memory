**2. Add `get_embeddings_for_doc_ids` to VectorTeleportHandler (crates/memory-service/src/vector.rs):**

```rust
/// Retrieve embeddings for a set of doc_ids (for supersession detection).
/// Returns a map of doc_id -> embedding vector. Missing entries are silently skipped.
pub fn get_embeddings_for_doc_ids(&self, doc_ids: &[String]) -> HashMap<String, Vec<f32>> {
    let mut embeddings = HashMap::new();
    for doc_id in doc_ids {
        if let Ok(Some(entry)) = self.metadata.find_by_doc_id(doc_id) {
            let index = self.index.read().unwrap();
            if let Ok(Some(vector)) = index.get_vector(entry.vector_id) {
                embeddings.insert(doc_id.clone(), vector);
            }
        }
    }
    embeddings
}
```

**3. Add supersession detection to StaleFilter (crates/memory-retrieval/src/stale_filter.rs):**

Add a new method that accepts optional embeddings:
```rust
/// Apply time-decay and supersession detection.
/// embeddings: optional doc_id -> embedding map for supersession.
/// If None or empty, only time-decay is applied.
pub fn apply_with_supersession(
    &self,
    results: Vec<SearchResult>,
    embeddings: Option<&HashMap<String, Vec<f32>>>,
) -> Vec<SearchResult> {
    if results.is_empty() || !self.config.enabled {
        return results;
    }

    let newest_ts = self.find_newest_timestamp(&results);
    let mut adjusted = match newest_ts {
        Some(ts) => self.apply_time_decay(results, ts),
        None => results,  // No timestamps at all -- pass through
    };

    // Apply supersession if embeddings available
    if let Some(embs) = embeddings {
        if !embs.is_empty() {
            self.apply_supersession(&mut adjusted, embs);
        }
    }

    adjusted.sort_by(|a, b| b.score.partial_cmp(&a.score).unwrap_or(std::cmp::Ordering::Equal));
    adjusted
}
```

Implement `apply_supersession`:
- Sort results by timestamp descending (newest first) for comparison
- For each result i (older):
  - Skip if already has "superseded_by" in metadata
  - Skip if kind is exempt (Constraint, Definition, Procedure, Preference)
  - For each result j (newer than i):
    - If both have embeddings, compute cosine similarity via dot product (vectors are pre-normalized by CandleEmbedder)
    - If similarity >= config.supersession_threshold (0.80):
      - Insert `"superseded_by": j.doc_id` into i's metadata
      - Multiply i's score by `(1.0 - config.supersession_penalty)` (i.e., 0.85)
      - Break inner loop (no transitivity -- each result superseded at most once)

Update the existing `apply()` method to delegate to `apply_with_supersession(results, None)` for backward compatibility.

Implement dot_product helper:
```rust
fn dot_product(a: &[f32], b: &[f32]) -> f32 {
    a.iter().zip(b.iter()).map(|(x, y)| x * y).sum()
}
```
