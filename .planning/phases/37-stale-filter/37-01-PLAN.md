**2. Create crates/memory-retrieval/src/stale_filter.rs:**

```rust
pub struct StaleFilter {
    config: StalenessConfig,
}
```

Implement:
- `pub fn new(config: StalenessConfig) -> Self`
- `pub fn apply(&self, results: Vec<SearchResult>) -> Vec<SearchResult>` - main entry point
  - If results empty or !config.enabled, return unchanged
  - Find newest_ts from results (max timestamp_ms in metadata)
  - Apply time-decay to each non-exempt result
  - Re-sort by score descending
  - Return adjusted results
- `fn find_newest_timestamp(&self, results: &[SearchResult]) -> Option<i64>` - parse "timestamp_ms" from metadata, return max
- `fn apply_time_decay(&self, results: Vec<SearchResult>, newest_ts: i64) -> Vec<SearchResult>` - for each result:
  - Parse timestamp_ms from metadata; if missing, skip (no penalty)
  - Parse memory_kind from metadata; default to "observation"
  - If kind is Constraint, Definition, Procedure, or Preference: skip (exempt)
  - Calculate age_days = (newest_ts - this_ts) as f64 / 86_400_000.0
  - Apply locked formula: `score * (1.0 - max_penalty * (1.0 - (-age_days / half_life).exp()))`
- `fn is_exempt(kind_str: &str) -> bool` - match against "constraint", "definition", "procedure", "preference" (case-insensitive)

**Important:** Apply uniform decay across all retrieval layers (per Claude's discretion -- keeping it simple). All Observations get full decay regardless of salience score (per Claude's discretion). No per-query skip_staleness flag (deferred).

The `apply()` method does NOT handle supersession yet -- that requires embeddings and will be added in Plan 37-02's wiring. Leave a clear `// TODO(37-02): supersession detection requires embeddings` comment.

fn build_metadata(timestamp_ms: Option<i64>, agent: Option<&str>, memory_kind: &str) -> HashMap<String, String> {
    let mut metadata = HashMap::new();
    if let Some(ts) = timestamp_ms {
        metadata.insert("timestamp_ms".to_string(), ts.to_string());
    }
    if let Some(a) = agent {
        metadata.insert("agent".to_string(), a.to_string());
    }
    metadata.insert("memory_kind".to_string(), memory_kind.to_string());
    metadata
}
