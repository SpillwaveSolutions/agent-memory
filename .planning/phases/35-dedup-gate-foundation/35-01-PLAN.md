  <name>Task 1: Create InFlightBuffer ring buffer in memory-types</name>
  <files>crates/memory-types/src/dedup.rs, crates/memory-types/src/lib.rs</files>
  <action>
Create `crates/memory-types/src/dedup.rs` with:

1. `BufferEntry` struct: `event_id: String`, `embedding: Vec<f32>`. Derive Debug, Clone.

2. `InFlightBuffer` struct with fields: `entries: Vec<Option<BufferEntry>>`, `capacity: usize`, `dimension: usize`, `head: usize`, `count: usize`. Derive Debug.

3. Implement `InFlightBuffer`:
   - `new(capacity: usize, dimension: usize) -> Self` -- pre-allocate `entries` as `vec![None; capacity]`
   - `push(&mut self, event_id: String, embedding: Vec<f32>)` -- assert `embedding.len() == self.dimension`, write to `entries[self.head]`, advance head with modular wrap, increment count up to capacity
   - `find_similar(&self, query: &[f32], threshold: f32) -> Option<(String, f32)>` -- iterate all `Some` entries, compute cosine similarity (dot product on normalized vectors), track best match above threshold, return `Some((event_id, score))` or `None`
   - `len(&self) -> usize` returns count
   - `is_empty(&self) -> bool`
   - `capacity(&self) -> usize`
   - `clear(&mut self)` -- set all entries to None, reset head and count to 0

4. Private `cosine_similarity(a: &[f32], b: &[f32]) -> f32` -- dot product (vectors are pre-normalized by CandleEmbedder)

5. Unit tests in `#[cfg(test)] mod tests`:
   - `test_push_and_find_exact_match` -- push an entry, query with same vector, expect score ~1.0 above threshold
   - `test_no_match_below_threshold` -- push entry, query with orthogonal vector, expect None
   - `test_ring_buffer_overwrites_oldest` -- push capacity+1 entries, verify oldest is overwritten and newest is findable
   - `test_empty_buffer_returns_none` -- find_similar on empty buffer returns None
   - `test_clear_resets_buffer` -- push entries, clear, verify is_empty and find_similar returns None
   - `test_dimension_mismatch_panics` -- use `#[should_panic]`, push embedding with wrong dimension

Update `crates/memory-types/src/lib.rs`:
- Add `pub mod dedup;` to module list
- Add `pub use dedup::{BufferEntry, InFlightBuffer};` to re-exports
